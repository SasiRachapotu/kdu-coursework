# Homework 1
### This homework consists of 4 modules:
#### 1. Core: This consist of Student class, this contains all the getter and setter functions, also done constructor overloading for other ways to create objects.
#### 2. Data: This consist of StudentRepository class where I have created a arrayList for the Student objects and added all necessary functions like add, retrieve, update and also demonstrated the method overloading in the update student function.
#### 3. logging: This consist of all the functions that make logs to the file myLogfile.log present in resources of logging module
#### 4. App: this consist of main function which integrates all modules and I have created a user driven program to demonstrate the functonality of the application.
